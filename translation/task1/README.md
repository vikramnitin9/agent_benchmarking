## Evaluation Criteria

User Experience:
- Stops when finished: 3
- Stops only when finished: 3
- Attempts to compile, run and test the code (irrespective of whether this attempt is successful or not): 4

Correctness:
- Produces compilable Rust code: 3
- Preserves compression functionality of original C program: 3
- Uses a native Rust translation, as opposed to using library implementations for compression, etc: 4
